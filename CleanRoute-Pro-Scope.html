1Project Overview

CleanRoute Pro is a custom-built route scheduling and optimisation platform designed specifically for cleaning companies. The application automates travel time and distance calculations between jobs using live Google Maps data, replacing the manual process of individually plugging addresses into Google Maps for each team, each day.

This scope covers the complete build: a production-ready SaaS platform with multi-tenant architecture, user authentication, subscription billing via Stripe, a saved client database, schedule templates for recurring 4-week rotations, staff rostering, and all features demonstrated in the beta preview.

2Features Included

2.1 Route Scheduling & Calculation

Feature Description
Base Address Set a starting and return point for each team with Google Places autocomplete. Base address is fully editable at any time
Client Scheduling Add clients with address, name, and configurable job duration. Supports manual input of total job hours including decimal values (e.g. 2.25, 1.5)
Per-Job Staff Count Assign a staff count to each job (1, 2, or 3+). Job duration is automatically split based on staff assigned (e.g. 4.5 hours with 3 staff = 1.5 hours each)
Live Travel Calculation Real-time travel time and distance between every stop via Google Maps Directions API
Traffic-Aware Timing Travel estimates are dynamically calculated based on the scheduled departure time for each leg of the route, reflecting actual time-of-day traffic conditions (e.g. 7:52am departure reflects morning traffic, 9:30am reflects mid-morning conditions)
Auto-Timed Schedule Start and end times for each client calculated automatically based on travel + job duration
Editable Client Start Time Lock a specific start time on any individual client (e.g. "Mrs Smith at 9:00am"). The rest of the day's schedule recalculates around the fixed time
Route Optimisation One-click button to reorder clients for the most efficient driving route (Google-powered)
Lock Job Position Lock individual jobs in place so they are excluded from route optimisation. Ideal for clients with fixed time slots
Reorder Jobs Move clients up and down in the schedule with one click; entire schedule recalculates instantly
Swap / Replace Client Change the address on an existing client card; schedule recalculates automatically
Cancel Client Remove a client from the schedule; all travel and times update instantly
Lunch / Breaks Insert configurable breaks between clients; factored into schedule timing
Interactive Map Live map showing colour-coded routes, numbered markers, and full route polylines
API Call Optimisation Smart caching of previously calculated routes, batching of API requests, and avoidance of redundant recalculations to minimise Google Maps API usage and costs

2.2 Multi-Team Support

Feature Description
Team Tabs Unlimited colour-coded teams, each with independent schedules, base addresses, and settings. Add as many teams as your business requires
Per-Team Settings Configurable hourly rate, day start time, and vehicle fuel settings per team
Team Breakdown Clear visibility of which staff are assigned to each team and total headcount per team

2.3 Financials & Reporting

Feature Description
Daily Summary KPI dashboard: total clients, job time, travel time, distance, total work hours
Wage Calculation Automatic wage output based on configurable hourly rate × total work time. Displayed in both standard format (2h 15m) and decimal format (2.25 hours) for payroll compatibility
Fuel Cost Calculator Calculate fuel costs based on total distance, vehicle efficiency (L/100km), and fuel price ($/L)
Per-KM Allowance Configurable per-kilometre rate (e.g. $0.99/km) as an alternative cost calculation alongside the fuel calculator
CSV Export Export the day's full schedule as a downloadable CSV file. Includes departure time, travel duration, arrival time, job start time, job end time, staff assignments, team headcount, and daily summary

2.4 Client Database

Feature Description
Saved Client List Add clients once and save them permanently. Includes name, address, email, phone number, default job duration, and notes
Quick Search & Add Search saved clients by name and add them to the schedule with one click
Edit / Remove Clients Update client details or remove from the database at any time

2.5 Schedule Templates

Feature Description
Save Weekly Templates Save a complete week's schedule as a reusable template (e.g. B1, A2, B3, A4)
Load & Edit Load a saved template when that week rolls around; make edits (swaps, cancellations) as needed
4-Week Rotation Supports full 4-week recurring schedule blocks with labelled templates

2.6 Account, Access & Multi-Tenant

Feature Description
User Authentication Secure login with email and password via Supabase Auth
Multi-Tenant Architecture Each business (customer) gets fully isolated data. Schedules, clients, and settings are completely separated
Admin & Staff Roles Admin accounts manage schedules and settings; staff accounts can view their assigned routes and team schedules
Cloud-Hosted Data All schedule, client, and template data saved to a secure cloud database (Supabase) with row-level security
Access from Any Device Fully responsive. Works on desktop, tablet, and mobile browsers
Google Maps Navigation Staff can tap any job in their team's route to open turn-by-turn navigation directly in Google Maps on their mobile device
Admin Panel Platform-level admin dashboard to view and manage all tenant accounts, monitor subscription status, and oversee platform usage

2.7 Subscription Billing (Stripe)

Feature Description
Stripe Integration Full payment processing via Stripe for subscription billing
Subscription Tiers Configurable pricing plans (e.g. Basic, Pro, Enterprise) for businesses subscribing to the platform
Billing Management Customers can manage their subscription, update payment methods, and view invoices via Stripe Customer Portal

2.8 Staff Rostering

Feature Description
Staff Directory Manage a list of team members with names, contact details, and roles
Team Assignment Assign staff members to teams on a per-day basis. Staff rosters can change day to day
Availability Mark staff as available or unavailable for specific days

2.9 Onboarding

Feature Description
Guided Setup Flow Step-by-step onboarding wizard for new users covering account creation, business details, and initial configuration
First Team & Client Setup Guided prompts to create their first team, set a base address, and add their first clients to the database
Welcome Overview Introductory walkthrough highlighting key features and navigation to help new users get started quickly

2.10 Checklists

Feature Description
Master Checklist Templates Create reusable checklist templates with customisable items (e.g. kitchen, bathrooms, windows). Templates can be edited, duplicated, or deleted at any time
Per-Client Customisation Assign a master template to a client and customise it for their specific requirements. Add, remove, or modify checklist items per client
Saved to Client Profile Each client's customised checklist is saved permanently to their profile in the client database and loads automatically when the client is scheduled
Staff Checklist View Staff can view and tick off checklist items on their mobile device when they arrive at a job
Completion Tracking Track whether a checklist was completed for each visit, providing visibility for admins on job quality and task completion
Email Checklist to Client Once staff complete a checklist, a summary can be automatically emailed directly to the client as a completed job report
